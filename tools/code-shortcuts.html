# 文件编辑

# 翻页查看 光标不移动
Command + PgUp 按行向上滚动
Command + PgDown 按行向下滚动

# 跳转
Ctrl + G 跳转至某行
Command + Shift + O 跳转到某个符号

# 跳转一个单词
Option + ←
Option + →

# 移动行
Option + Up 向上移动行
Option + Down 向下移动行

# 复制行
Option + Shift + Up 向上复制行
Option + Shift + Down 向下复制行

# 插入
Command + Enter 下一行插入
Command + Shift + Enter 上一行插入

# 缩进
Command + [ 减少缩进
Command + ] 增加缩进

# 格式化
Shift + Option + F 格式化代码
