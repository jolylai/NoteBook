# 元素尺寸

  • 元素尺寸：包含 padding 和 border 即 border-box 的尺寸， js 中为 offsetWidth 和 offsetHeight
  • 元素内部尺寸：包含 padding 但不包括 border 的尺寸，js 中为 clientWidth 和 clientHeight
  • 元素外面尺寸： 包含 padding 、border 和 margin, 没有原生 DOM api
