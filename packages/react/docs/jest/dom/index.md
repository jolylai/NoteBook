## DOM

> [Custom jest matchers to test the state of the DOM](https://github.com/testing-library/jest-dom)

安装

```bash
yarn add @testing-library/jest-dom --dev
```

使用

测试配置文件引入

```js
import '@testing-library/jest-dom';
```
